# Think Tank - Website 


**Clone Repository** 

```
$ git clone https://github.com/Think-Tank-ITS/website.git
$ cd website
```

**Install Dependencies**

```
$ npm install -f pnpm (if you haven't installed pnpm)
$ pnpm install

```

**Run Project**

```
$ pnpm run dev
# open the site at http://127.0.0.1:3000
```
